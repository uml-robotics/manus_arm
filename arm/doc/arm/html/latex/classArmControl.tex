\subsection{\-Detailed \-Description}
\-Arm control node. 

\-This node receives movement commands from teleop nodes and moves the arm hardware appropriately. \-The operation of the node is autonomous; after the \doxyref{\-Arm\-Control}{p.}{classArmControl} object has been created, the node will run with no further operation.
