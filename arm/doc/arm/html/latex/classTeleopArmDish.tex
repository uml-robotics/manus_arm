\subsection{\-Detailed \-Description}
\-Node for \-A\-R\-M teleop from the neuron dish. 

\-This node receives \-C\-A\-T (center of activity trajectory) data from the \-C\-A\-T creator node and then creates and publishes movement commands. \-The operation of the node is autonomous; after the object has been created, the node will run with no further operation.
